    About

    Named after a Latin word meaning ‘I translate’, Traduco is a simple yet powerful translation tool. Open either a plain text file (.txt, for example) to start a translation, or a project (stored in .json) to continue the work. While translating, you can make and consult notes using the text input field at the top of the screen. At the bottom of the screen, you will find source and target texts' symbol count in the following format: selection / parapgraph / whole text (proportion of the target text's length to that of the source one in %). To make the process easier, Traduco offers a number of hotkeys:

    • Alt+left arrow - select previous cell
    • Alt+right arrow - select next cell
    • Alt+up arrow - select cell above
    • Alt+down arrow - select cell below
    • Alt+Page Up - go to the previous page
    • Alt+Page Down - go to the next page
    • Alt+Home - switch between notes and current cell
    • Alt+End - switch between statistics and current cell
    • Alt+Insert - copy source to target
    • Alt+Delete - clear current cell
